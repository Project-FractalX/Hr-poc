# FractalX Generated Microservices

This directory contains microservices generated by FractalX.

## Services

- **department-service**: http://localhost:8085
- **employee-service**: http://localhost:8081
- **leave-service**: http://localhost:8083
- **payroll-service**: http://localhost:8082
- **recruitment-service**: http://localhost:8084
- **fractalx-saga-orchestrator**: http://localhost:8099

## Quick Start

```bash
./start-all.sh
```

### Start services individually

```bash
# Terminal 1 - Start department-service
cd department-service
mvn spring-boot:run
```

```bash
# Terminal 2 - Start employee-service
cd employee-service
mvn spring-boot:run
```

```bash
# Terminal 3 - Start leave-service
cd leave-service
mvn spring-boot:run
```

```bash
# Terminal 4 - Start payroll-service
cd payroll-service
mvn spring-boot:run
```

```bash
# Terminal 5 - Start recruitment-service
cd recruitment-service
mvn spring-boot:run
```

## Service Architecture

```
Standalone Services (No Service Discovery)
└── Static Gateway Routing
    ├── department-service (:8085)
    ├── employee-service (:8081)
    │   └─ Dependencies: DepartmentService, PayrollService
    ├── leave-service (:8083)
    │   └─ Dependencies: PayrollService, EmployeeService
    ├── payroll-service (:8082)
    ├── recruitment-service (:8084)
    │   └─ Dependencies: DepartmentService, EmployeeService
    └── Gateway → http://localhost:9999
```
